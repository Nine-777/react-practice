# React Practice App
React + Vite + TypeScript

## 主なディレクトリ構成
```sh
public/               #... 処理にかけないファイルの収納ディレクトリ
src/
  ├── main.tsx        #... エントリーポイント（React実行開始ファイル）
  ├── App.tsx         #... ルートコンポーネント
  ├── pages/          #... ページコンポーネントディレクトリ
  │   ├── Home.tsx    #...... 投稿一覧ページ
  │   └── post.tsx    #...... 投稿詳細ページ
  ├── components/     #... コンポーンネントディレクトリ
  └── assets/         #... 素材置場ディレクトリ
      └── images/     #....... 画像ファイルディレクトリ
```
